# AWS AI Practitioner Challenge

Projects completed as part of the AWS AI Practitioner Challenge through the Udacity AWS Agentic Engineer Nanodegree.

This challenge covers core AI/ML concepts, generative AI technologies, responsible AI principles, AWS AI services, prompt engineering, and hands-on application building.

## Projects

| # | Project | Description | Status |
|---|---------|-------------|--------|
| 1 | [IT Helpdesk Triage Support Tool](./project-1) | A PartyRock app that triages IT helpdesk issues into structured support tickets and troubleshooting guides | ✅ Complete |
| 2 | [Air Quality Data Analysis](./project-2) | AI-powered analysis of a 30-day air quality dataset using PartyRock's Analyze Data feature, with evaluation of AI accuracy and limitations | ✅ Complete |

## Technologies and Services Covered

- **AWS Services:** Amazon Bedrock, PartyRock, SageMaker, Rekognition, Lex, Polly, Transcribe, Translate, Comprehend, Amazon Q
- **Concepts:** Foundation models, prompt engineering, RAG, fine-tuning, responsible AI, model evaluation and optimisation
- **Tools:** PartyRock App Builder (Amazon Bedrock Playground), PartyRock Data Analysis (Whiskers)

## Course Context

The AWS AI Practitioner Challenge is a component of the AWS Agentic Engineer Nanodegree offered through Udacity. It provides hands-on experience building AI-powered applications and analysing data using AWS services while developing a practical understanding of generative AI principles and responsible AI design.
